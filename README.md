## ğŸ•µï¸ Source Analysis Mode

When using default sources, you can enable **source analysis mode** to evaluate the quality of every proxy source. This helps identify:

- **Dead sources** that return zero working proxies
- **Redundant sources** where all proxies are already covered by other sources
- **Overlapping pairs** that likely scrape from each other

The analysis outputs a detailed table for each source:

| Column | Description |
|--------|-------------|
| **TOTAL** | Total proxies the source provides |
| **UNIQ** | Proxies found only in this source |
| **WORK** | Working proxies from this source |
| **W.UNIQ** | Working proxies exclusive to this source |
| **STATUS** | `KEEP` (2+ working) or `REMOVE` (<2 working) |

Results are saved to `source_analysis.json` for further review.

## ğŸ”§ Proxy File Format

Your proxy file should contain one proxy per line:

```
192.168.1.1:8080
http://proxy.example.com:3128
http://user:password@proxy.example.com:8080
https://secure-proxy.example.com:443
```

Supported formats:

- `ip:port`
- `http://ip:port`
- `https://ip:port`
- `http://user:pass@ip:port`
